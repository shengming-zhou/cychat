# CyChat

CyChat is an experimental small Chinese-language model training project. It uses
a three-stage workflow: local development on an M4 Mac, CUDA validation on one
RTX 5090, and a final training run on 8 H100 GPUs with an approximately two-hour
wall-clock budget.

Project constraints and the current cross-conversation handoff are maintained in
[`README_BY_AGENT.md`](README_BY_AGENT.md).
