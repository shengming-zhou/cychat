# Agent instructions

Before working on this repository, read `README_BY_AGENT.md` completely. Treat
its project goals, constraints, decisions, and current status as the durable
handoff between conversations.

Keep `README_BY_AGENT.md` concise and current whenever a durable decision is
made, an experiment changes the project status, or the next action changes.
Do not store chat transcripts, credentials, or speculative claims in it.

The local `nanochat/` directory is reference material only. Never add, commit,
or push it (or copied repository metadata) to CyChat.

The user is the primary code author. Do not implement or rewrite project source
code unless the user explicitly asks; prefer explanation, review, debugging,
pseudocode, and experiment design.
